# Reaction Aim
Reaction Aim is a minimalist arcade game and reaction trainer made with raylib. I do not intend it as an advance tool as this is just a side project of mine.

## Installation
You can download pre-built binaries from the releases page. Alternatively, you can build it yourself if you want to make changes.

## Building from Source
1. Make sure you have a C compiler, raylib, and CMake installed.
2. Download this repo or clone it.
3. Initialize the project with CMake.
```
cmake -S . -B build
```
4. Compile the program.
```
cmake --build build
```
5. Finally, run the program found in `build/`.
